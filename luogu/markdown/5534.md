# 【XR-3】等差数列

## 题目描述

小 X 给了你一个等差数列的前两项以及项数，请你求出这个等差数列各项之和。

等差数列：对于一个 $n$ 项数列 $a$，如果满足对于任意 $i \in [1,n)$，有 $a_{i+1} - a_i = d$，其中 $d$ 为定值，则称这个数列为一个等差数列。

## 输入输出格式

### 输入格式

一行 $3$ 个整数 $a_1, a_2, n$，表示等差数列的第 $1,2$ 项以及项数。

**数据范围：**

- $|a_1|,|a_2| \le 10^6$。

- $3 \le n \le 10^6$。

### 输出格式

一行一个整数，表示答案。

## 输入输出样例

### 输入样例 #1

```cpp
1 2 3

```
### 输出样例 #1

```cpp
6

```
### 输入样例 #2

```cpp
-5 -10 5

```
### 输出样例 #2

```cpp
-75

```
## 说明

【样例 $1$ 说明】

这个等差数列为 `1 2 3`，其各项之和为 $6$。
