Lining Up

题意翻译

## 题目描述 在平面直角坐标系中给定 $n$ 个点，现画一条直线，问最多可以经过多少个点。 ## 输入输出格式 ### 输入格式 输入包括多组数据。 输入的第一行为一个正整数 $T$，表示测试数据组数。接下来T组测试数据，每组测试数据包括多行，每行为两个整数 $x, y$ ，表示一个点 $(x,y)$ 。每组测试数据之前有一个空行。 ### 输出格式 对于每组数据，输出一行一个正整数，表示最多可以经过的点的数量。每两组输出之间用一个空行分割。 ## 输入输出样例 **输入样例#1：** ``` 1 1 1 2 2 3 3 9 10 10 11 ``` **输出样例#1：** ``` 3 ``` ## 说明 $ 1 \leq \text{点数} \leq 700 $
