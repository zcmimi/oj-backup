Meta-Loopless Sorts

题意翻译

program sort(input,output);
var
a,b,c : integer;
begin
readln(a,b,c);
if a &lt; b then
if b &lt; c then
writeln(a,b,c)
else if a &lt; c then
writeln(a,c,b)
else
writeln(c,a,b)
else
if a &lt; c then
writeln(b,a,c)
else if b &lt; c then
