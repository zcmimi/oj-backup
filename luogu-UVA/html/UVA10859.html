Placing Lampposts

题意翻译

# 题面描述 给定一个$n$个点$m$条边的无向无环图，在尽量少的节点上放灯，使得所有边都与灯相邻（被灯照亮）。 在灯的总数最小的前提下，被两盏灯同时照亮的边数应该尽可能大。 # 输入格式 第一行输入$T$,为数据组数。 每组数据第一行输入$n,m$，分别为该组数据中图的点数和边数。 以下$m$行，输入各边的两端点$u,v$。 # 输出格式 输出共$T$行。 对每组数据，一行输出三个数，最小灯数、被两盏灯同时照亮的边数、只被一盏灯照亮的边数。
