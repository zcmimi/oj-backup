救济金发放 The Dole Queue

题意翻译

题目描述 n(n<20)个人站成一圈，逆时针编号为 1~n。有两个官员，A从1开始逆时针数，B从n开始顺时针数。在每一轮中，官员A数k个就停下来，官员B数m个就停下来（两个官员有可能能停在同一个人上）。接下来被官员选中的1个或2个人离开队伍。 输入格式 输入n ,k ,m ，可能有多组数据，以 0 0 0结尾。 输出格式 输出每轮里被选中的人的编号（如果有两个人，先输出被A选中的）。输出的每个数应正好占3列。样例中的“ ␣ ”代表一个空格。
