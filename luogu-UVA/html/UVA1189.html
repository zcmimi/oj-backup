Find The Multiple

题意翻译

## 题目描述 给出一个整数n，(1 <= n <= 200)。求出任意一个它的倍数m，要求m必须只由十进制的'0'或'1'组成。 ## 输入输出格式 ###### 输入格式： 输入文件可能包含多个测试样例。每行包含一个 n (1 < = n < = 200) 。包含零的行将终止输入。 ###### 输出格式： 对于输入中的n , 输出对应的 m 。m 用十进制表示，不能超过100位。如果 n 有多个解决方案, 则其中任何一个都是可以的。 ## 输入输出样例 ###### 输入样例： 2 6 19 0 ###### 输出样例： 10 100100100100100100 111111111111111111 感谢@zhuyifan 提供的翻译
