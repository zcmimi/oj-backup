无关的元素 Irrelevant Elements

题意翻译

对于给定的n个数，a1,a2,……,an，依次求出相邻两数之和，将得到一个新数列。重复上述操作，最后结果变成一个数。问这个数除以m的余数与哪些数无关？例如n=3，m=2时，第一次求和得到a1+a2，a2+a3，再求和得到a1+2a2+a3，它除以2的余数和a2无关。(1 ≤ n ≤ 1e5, 2 ≤ m ≤ 1e9 )
