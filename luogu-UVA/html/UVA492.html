Pig-Latin

题意翻译

``` 对每一个单词做如下处理： 1： 如果这个单词以元音字母开头（a，e，i，o，u&&A，E，I，O，U） 在单词后面加上“ay”； 如“a”-“aay” 2： 如果这个单词以辅音字母开头（不是元音的字母）把第一个字母放到最后在加上“ay” 例如“hello”-“ellohay” 3： 不要改变任何字母的大小写！ ``` ### 输入样例#1： This is the input. ### 输出样例#1： hisTay isay hetay inputay. _注意句末的句号_
