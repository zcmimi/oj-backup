Pesky Palindromes

题意翻译

题目描述 如果一个字符串从左到右读和从右到左读是一样的，那么我们说这个字符串是“回文字符串”。 现在你的任务是写一个程序，这个程序可以读入一个字符串，然后求出有多少个不同的子串是回文字符串。 输入输出格式 输入格式 输入文件由多个字符串组成，一行一个。每个字符串长度不超过80个字符。字符串前面没有空格。 输出格式 对于每个非空的行，输入一行信息： The string 'input string' contains nnnn palindrones. 其中，用读入的字符串代替'input string',用求出的子串个数代替'nnnn'。 感谢@2017_9_gaoZhiYuan 提供的翻译
