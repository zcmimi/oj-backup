# Peach Blossom Spring

## 题意翻译

题目大意：

1到k中分别有一个人，n-k+1到n是房子，要求每个人通过修复道路都可以到达一座房子，且一座房子只能容纳一人，前k个点和最后k个点一一对应。

求最小修复费用；无解输出No solution
