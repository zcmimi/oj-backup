# Crazy tea party

## 题意翻译

$n$个人按编号围着桌子坐成一圈，编号由$1-n$，每次只能交换相邻座位的人的位置，问多少次可以将所有人的左右两人互换，即左边的人到他的右边，右边的人到他的左边。(即座位顺序从原来的顺时针方向变为逆时针方向)。
