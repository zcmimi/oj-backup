# 循环小数 Repeating Decimals

## 题意翻译

输入整数$a$ 和$b$ （0≤$a$ ≤3000,1≤$b$ ≤3000），输出a/b的循环小数表示以及循环节的长度。例如$a$ =5，$b$ =43

小数表示为0.(116279069767441860465)，循环节长度为21。

补充：

如果循环节超过50位，就在第50位后打省略号（三个点，就像“…”）
