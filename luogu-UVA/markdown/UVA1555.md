# Garland

## 题意翻译

**问题描述**

有图

![](http://61.139.95.227:82/upload/image/20180704/20180704210608_97062.png)

并且满足：

H1 = A

Hi = (H[i−1] + H[i+1])/2 − 1,i∈(1,N)

HN = B

Hi ≥ 0,i∈[1,N]

（H即Height）

求给定最左边的A点和个数N，求最右边B点的最低值。

输入格式：两个数，分别表示N,A

整数N (3 ≤ N ≤ 1000)，实数A (10 ≤ A ≤ 1000)

**Sample input**

692 532.81

**Sample Output**

446113.34
