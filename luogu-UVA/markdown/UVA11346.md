# 概率 Probability

## 题意翻译

在$[-a,a]*[-b,b]$的区域内（其实就是一个矩形内）取一个点P，连接$OP$

请求出以$OP$为对角线的矩形面积大于S的概率(O为原点)

$$a,b>0,s>=0$$
