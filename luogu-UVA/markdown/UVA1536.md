# Coins

## 题意翻译

题目描述：

称硬币问题

给定N枚硬币，其中N-1枚为真币，1枚为假币，真币们重量都一样，假币与真币重量不同。给出一架没有砝码的天平，问称K次能否找到那枚假币？

输入格式：

第一行为数据组数T。之后T行，每行两个数N，K，2≤N≤100，0≤K≤100。

输出格式：

对于每一对（N,K），如果能找到假币，输出一行“POSSIBLE”；如果不能，输出一行“IMPOSSIBLE”。
