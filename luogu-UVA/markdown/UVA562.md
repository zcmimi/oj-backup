# Dividing coins

## 题意翻译

# 题目描述

给你一堆硬币，让你分成两堆，分别给A，B两个人，求两人得到的最小差。

# 输入

输入有多组测试样例。第一行一个数字n，代表有n组测试样例。接下来是n组测试样例，每个样例包括两行，第一行m代表有m个硬币，第二行输入m个硬币的面值。

# 输出

输出每组样例的最小差值。
