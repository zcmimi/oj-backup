# 快速幂计算 Power Calculus

## 题意翻译

```plain

读入 $n$ $(1\le n\le 1000)$，问最少几次乘除法可以从$x$得到$x^n$，例如$x^{31}$需要6次：$x^{2}$ $=$ $x\cdot x$，$x^{4}$ $=$ $x^{2}\cdot$ $x^{2},$ $\dots ,$ $x^{32}$ $=$ $x^{16}\cdot x^{16},$ $x^{31}$ $=$ $x^{32}÷x$

```
