# Concert Hall Scheduling

## 题意翻译

给出$n$个乐队演奏的开始日期$x$，结束日期$y$，愿意支付的费用$w$，试选取一些乐队，使每个时间点最多有两支乐队在演奏，且支付的费用最大

注：日期最多为$365$，输入可能有多组数据，以一个$0$结束输入
