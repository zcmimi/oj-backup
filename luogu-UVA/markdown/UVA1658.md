# 海军上将 Admiral

## 题意翻译

`翻译来自紫书`

给出一个$v(3\leq v\leq 1000)$个点$e(3\leq e\leq 10000)$条边的有向加权图，求$1$~$v$的两条不相交（除了起点和终点外没有公共点）的路径，使得权和最小。如（$PDF$中）图所示，从$1$到$6$的两条最优路径为$1-3-6$（权和为$33$）和$1-2-5-4-6$（权和为$53$）。
