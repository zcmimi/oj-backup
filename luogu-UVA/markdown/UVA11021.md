# Tribles

## 题意翻译

### 题目大意

一开始有$k$种生物，这种生物只能活1天，死的时候有$p_i$的概率产生$i$只这种生物（也只能活一天），询问m天内所有生物都死的概率（包括m天前死亡的情况）

### 输入格式

第一行输入一个整数$T$，表示数据总数

每一组先输入三个整数$n(1<=n<=1000),k(0<=k<=1000),m(0<=m<=1000)$

然后输入n个整数，分别为$p_0$到$p_{n-1}$

### 输出格式

对于每一组数据，先输出"Case #x: "

再输出答案，精度要求在1e-6以内
