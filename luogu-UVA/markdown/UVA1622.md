# 机器人 Robot

## 题意翻译

有一个N*M的网格，每个网格内有一个机器人，每次可以发出一个指令，指令只会是以下之一：NORTH、SOUTH、EAST、WEST。每条指令可以使所有机器人向该方向走一格。如果一个机器人走出了网格，它将会炸毁（也就是说不再接受任何命令），现在给出4条指令的总条数，使所有机器人执行的指令条数和最大，保证答案在2^63-1内。
