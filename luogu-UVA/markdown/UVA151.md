# Power Crisis

## 题意翻译

1，2，...,N排成一个圈，让数字依次出列，1最先出列，之后每隔m个数便出列一个数。例如m=5,则1之后下一个出列的数字就是6. 现在给定N,请算出最小的m，使得13号是最后一个出列的数字。
