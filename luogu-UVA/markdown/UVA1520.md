# 航班 Flights

## 题意翻译

一场军事演习昨天开始了。其中炮兵发射导弹，而航空则向步兵提供补给。

#### 整个军事地面空间是一条直线。航空基地和步兵团就在这条线上的某个地方

大炮到处发射弹道导弹（李云龙：二营长，我的意大利炮呢）。

### 所有的导弹发射都是有计划的，每次都是在特定的时间沿着特定的轨道发射的。航空飞行也计划在一定的时间和空间间隔。

但是有那些导弹，即使在演习中也可能致命！

### 你应该帮助航空将军计划每一次飞行的最低安全高度。

### 考虑到飞行的时间间隔和空间间隔信息，飞行的最小安全高度是最小高度，使得相应时间间隔和空间间隔内的所有导弹轨迹都处于或低于该高度。如果在飞行的时间和空间间隔内没有导弹，那么最小安全高度定义为零。弹道导弹从地面发射，地面的高度定义为零，并沿垂直对称抛物线飞行。

导弹速度忽略了这一问题，假设导弹瞬间跟随其弹道。例如，下图显示了实线中两个弹道导弹的轨迹，以及虚线中四个不同飞行的最小安全高度。垂直线界定了此示例中每个航班的空间间隔。本样本中的飞行时间间隔包括发射两枚导弹。![](https://cdn.luogu.org/upload/pic/53604.png)

# input

第一行输入包含一个整数n——计划发射的导弹数量（1≤n≤50000）
