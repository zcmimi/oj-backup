# 汉堡 Burger

## 题意翻译

设一共有$n$个汉堡（$n$为偶数且$n<=100000$），其中有$n/2$个火腿汉堡，$n/2$个芝士汉堡。

$n$个小朋友坐成一排，依次等概率抽选汉堡。有意思的是，抽完倒数第三个小朋友时，厨师发现抽不下去了，因为正好剩下了两个相同的汉堡。

厨师觉得很有趣，想问你出现这种情况的概率。

输入格式

第一行一个整数$t$，表示测试组数。

对于每组测试，输入一个$n$，意义如上。

输出格式

输出$t$行，每行为对应测试的答案。
