# 收购游戏 Takeover Wars

## 题意翻译

两个集团在进行收购战争，每次每个集团可以采用两种收购操作， 1.用自己的子公司a收购自己另外的子公司b，完成之后b公司消失，a公司价值变成val[a]+val[b], 2.用自己的子公司a收购对方的子公司c，必须满足val[a]>val[c]，完成之后c公司消失，a公司价值不变 双方轮流操作，直到某一方没有子公司存在则输。
