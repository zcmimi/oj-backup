# Network

## 题意翻译

【问题描述】

给你n个点及边的连接关系，求无向图的割点。

【输入】

有多组数据，第一行为n个点，接下来为多行（最多n行），每一行第一个为起点，后面紧跟着的为终点，起来和终点互相连同（如5 1 3 4，5和1连接，5和3连接，5和4连接），0表示这组数据结束。如果n为0表示程序结束。

【输出】

输出每组数据的割点数目。
