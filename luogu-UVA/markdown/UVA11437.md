# Triangle Fun

## 题意翻译

如图所示，在三角形ABC边上有三个点D、E、F，它们满足BD=BC/3，CE=CA/3，AF=AB/3。连接AD、CF，交于点R。连接AD、BE交于点P。连接BE、CF交于点Q。求三角形PQR的面积并四舍五入到整数。

输入第一行是一个整数n，表示有n组数据。

每组数据6个实数Ax,Ay,Bx,By,Cx,Cy

输出共n行，每行一个整数，表示三角形PQR的面积

感谢 @ Ebola 提供的翻译。
