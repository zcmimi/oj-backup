# 因特网带宽 Internet Bandwidth

## 题意翻译

给出点数$n$、源点$S$、汇点$T$、边数$m$，然后给出每条边的起点、终点、和容量，求出最大流。

输入有多组数据，以一个$0$结尾，**每组数据的答案之间有一个空行**，输出格式见样例
