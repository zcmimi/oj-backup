# Hypertransmission

## 题意翻译

有n个星球，坐标x,y,z，播放着1或0类节目，现在求一个半径，表示星球的接受信号半径，如果星球接受不同类型节目多于同类型节目，则该星球为间谍星球，现在求让最多星球变成间谍星球且半径要尽可能小。求个数和半径。
