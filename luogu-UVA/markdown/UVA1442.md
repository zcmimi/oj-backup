# 洞穴 Cav

## 题意翻译

一个洞穴的宽度为 $n$ ( $n\le10^6$ )个片段组成。已知位置[i,i+1]处的地面高度 $pi$ 和顶的高度 $si$ ( $0\le pi<si\le1000$ ),要求在这个洞穴里储存尽量多的燃料，使得在任何位置燃料都不会碰到顶(但可以无限接近)。 来源：刘汝佳《算法竞赛入门经典》 感谢@lyclyc 搬运翻译
