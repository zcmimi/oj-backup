# YAPTCHA

## 题意翻译

求出下列式子的答案

![](http://acm.hdu.edu.cn/data/images/con181-1009-1.JPG)

**【输入】**

第一行一个数字T，表示数据组数

第2到T+1行，每一行包括一个数字n

40%的数据：T≤10^3，1≤n≤10^3

100%的数据：T≤10^3，1≤n≤10^6

**【输出】**

对于每一组询问，输出答案。
