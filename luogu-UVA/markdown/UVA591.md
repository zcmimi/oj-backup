# Box of Bricks

## 题意翻译

有一些砖块堆砌成的列，给定每列砖中砖块的个数。由于砖块个数不同，导致某些列砖的高矮不同。问最少移动多少块砖，才能使得每一列砖的高矮都完全一样？
