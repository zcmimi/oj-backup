# 糖果 Candy

## 题意翻译

有两个盒子各有n (n<=2e5) 个糖，每天随机选一个（概率分别为p，1-p），然后吃一颗糖。直到有一天，打开盒子一看，没糖了！输入n，p，求此时另一个盒子里糖的个数的数学期望。

来自：《算法竞赛入门经典（第二版）》刘汝佳
