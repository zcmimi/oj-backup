# GCD Guessing Game

## 题意翻译

给定一个正整数n

要求猜一个（1～n）之间的数字p

每次可以猜一个数，不妨叫x

然后呢会告诉你x和p的最大公约数

要求使用最优策略，使最坏情况下猜数次数最少

输出最少次数

比如n=6的时候可以直接猜6，若回答1，则p为1或5，再猜一次就好了，为2，就是2或4，为3就是3，为6就是6

所以最坏情况是2
