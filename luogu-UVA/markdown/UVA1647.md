# 计算机变换 Computer Transformation

## 题意翻译

首先给你一个数字1，然后每次操作把1变成01，把0变成10，求n次操作后00的个数，（0 < n < 1000）

比如第一次操作后（n = 1时）序列变成01，第二次操作后（n = 2）序列变成1001
