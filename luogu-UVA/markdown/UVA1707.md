# Surveillance

## 题意翻译

## 题目描述

给你一个长度为len的环，以及n个区间，要你选择尽量少的区间，使得它们完全覆盖整个环。问最少要多少个区间。

## 输入格式

输入数据的第一行是两个整数len和n，代表环的长度以及区间个数。之后n行描述的是n个区间，每个区间分别用一对数字(a,b)表示，若a≤b则表示这个区间覆盖的是[a,b]部分，否则表示这个区间覆盖的是除掉[a+1,b-1]以外的其他部分。

## 输出格式

输出只有一行，一个整数，代表覆盖整个环所需要的最少区间个数。

## 样例

### 样例输入

100 7

1 50

50 70

70 90

90 40

20 60

60 80

80 20

### 样例输出

3

## 数据范围与提示

3 ≤ n ≤ 1e6

1 ≤ k ≤ 1e6

1 ≤ ai,bi ≤ n
