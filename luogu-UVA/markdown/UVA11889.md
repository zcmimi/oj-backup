# Benefit

## 题意翻译

输入第一行t,为数据组数。

接下来t行，每行两个数a,c。

每组数据输出一行一个整数，为使得lcm(a,b)=c成立最小的b，其中lcm为最小公倍数。若没有满足条件的b，输出“NO SOLUTION”。

数据范围：

1<=t<=10^5，1<=a,c<=10^7
