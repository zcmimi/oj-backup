# Factors and Factorials

## 题意翻译

因子与阶乘

输入一个正整数N (2<=n

例如，825可用数列（0 1 2 0 1）表示，意思是0个2，1个3，2个5，0个7和1个11.

Input

多组数据。每组数组包括一个正整数N。

输入以一个单独的0结尾。

Output

分别对每组数据进行输出。

每组数据各占一行。

每组输出以“N! = ”开始。N与题目中所述一致。

输出数列时，每两个数字间用空格隔开。

Sample Input

5

53

0

Sample Output

5! = 3 1 1

53! = 49 23 12 8 4 4 3 2 2 1 1 1 1 1 1

1
