# Equipment

## 题意翻译

给出n个5元组，从中选出k组，使得这些组中5个位置，每个位置上最大数之和最大，求这个和。

输入：

(输入有多组数据)

第一行T为数据组数，每组数据的第一行为n,k,接下来n行每行5列，表示n个五元组。

输出：对于每组数据，输出选出k个五元组中每五个位置上最大数之和的最大值。
