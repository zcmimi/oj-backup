# Big Number

## 题意翻译

~~题目背景没有营养，~~大意为求一个大数（很大很大）的阶乘的位数（即这个数的阶乘是几位数）

## 输入：

由数行整数组成。第一行包含一个整数n，表示有n组数据，接着n行，每行一个整数N（小于10的7次方的正整数）。

## 输出：

输出n行,每行是输入中出现的整数的阶乘中的位数。
