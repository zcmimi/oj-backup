# Network

## 题意翻译

多组数据读入一个数n，n=0时退出，代表n个点的无向图。下面若干行，每行第一个数为u，u=0时退出，后面若干个数v，代表和u相连的点v，遇到换行退出。对于每组数据，输出图的割顶个数（输入较坑，所以详细说明）。
