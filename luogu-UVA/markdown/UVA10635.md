# Prince and Princess

## 题意翻译

Description

给你两条路径，对于每条路径上的点各不相同，请你求出两条路径最长公共部分的长度。

Input

第一行是数据组数t

每组数据的第一行包含三个数，n,p,q。其中路径上的点的大小不会超过n^2.

第二行包含p+1个数，表示第一条路径

第三行包含q+1个数，表示第二条路径。

Output

见样例。输出最长路径长度。
