# Palinwords

## 题意翻译

给出一系列全部是大写字母的单词，寻找其中符合下列条件的单词：

1、包含两个长度至少为3的回文字符串str1 与 str2.

2、str1与str2不能互相包含，内容也不能完全一样。

按照单词出现顺序输出单词，每行一个。

感谢@dogfar 提供的翻译
