# Glass Beads

## 题意翻译

题意：

有N组数据，每组给你一串字符串，但是这串字符串是环形的，让你找个位置切开，使得它的字典序最小，输出切开的位置（如果答案不唯一，输出最小位置）

每个字符串均由小写字母组成，每组字符串长度最长为10000
