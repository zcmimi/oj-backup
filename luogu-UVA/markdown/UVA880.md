# Cantor Fractions

## 题意翻译

# 重题1014：

Cantor表是这样的一个表：

1/1(i=1),1/2(i=2),1/3(i=6),1/4(i=7),1/5,…

2/1(i=3),2/2(i=5),2/3(i=8), 2/4,…

3/1(i=4),3/2(i=9),3/3,…

4/1(i=10),4/2,…

5/1,…

（表来自1014）

现在要求对于每个i,求出对应的第i个分数（不要约分）

#### 输入

每一行一个i，含义同上

#### 输出

对于每一个i，输出一个分数。
