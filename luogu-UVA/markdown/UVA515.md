# King

## 题意翻译

题目翻译：有ｎ个数，ｍ个限制条件。每一个限制条件形如：１．x y gt c：表示ax + ax+1 + … +ay > c。２．x y It c：表示ax + ax+1 + …… +ay < c。有解输出“lamentable kingdom”，否则输出“successful conspiracy”。
