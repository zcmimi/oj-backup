# Error Curves

## 题意翻译

给定n个二次函数Si(x)=ax^2+bx+c (1<=i

定义F(x)=max(Si(x)) 0<=x<=1000

求F(x)的最小值

输入一共T组数据

输出保留4位小数

$a \leq 100, \; |b| \leq 5000, \; |c| \leq 5000;$

$T < 100, \;$输入均为整数
