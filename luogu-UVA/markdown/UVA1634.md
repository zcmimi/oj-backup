# 野餐 The Picnic

## 题意翻译

输入m(m≤100）个点，选出其中若干个点，以这些点为顶点组成一个面积最大的凸多边形，使得内部没有输入点（边界上可以有）。输入点的坐标各不同，且至少有三点不共线。
