# Open Credit System

## 题意翻译

大致题意

给一个长度为n的整数序列a0 a1 a2....an-1,找出两个整数ai和aj（i<j） 使得ai-aj最大 输入 第一行 组数 T 每组数据 第一行输入数据数量n(2<=n<=1e+5) 接下来是n个不超过150000的整数 输出 对于每组数据，输出最大ai-aj 感谢@Himself65 提供的翻译
