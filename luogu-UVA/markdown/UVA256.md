# Quirksome Squares

## 题意翻译

题意简述: 每一次输入一个数n（2,4,6,8），求所有的n位数使得这个数前一半+后一半的平方等于这个数。（可以有前导零）

例：(30 + 25)*(30 + 25) = 3025
