# Alice and Bob

## 题意翻译

bzoj的翻译：

Alice和Bob两个好朋友又开始玩取石子了。

游戏开始时，有N堆石子排成一排，然后他们轮流操作（Alice先手），每次操作时从下面的规则中任选一个：

·从某堆石子中取走一个、

·合并任意两堆石子

不能操作的人输。Alice想知道，她是否能有必胜策略。
