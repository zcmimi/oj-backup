# 魔法GCD Magical GCD

## 题意翻译

给一个长度为n(n≤100000)的数列(每个数a[i]≤10^12)，找到一个连续子序列使得子序列的公约数与长度的乘积最大，求这个最大值。共T组数据。

输入格式：

第一行：T

接下来每一组数据：

第一行为n，第二行包含n个正整数a[i]，其中a[i]≤10^12

输出格式：

一行，为子序列公约数与长度的乘积
