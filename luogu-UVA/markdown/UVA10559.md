# 方块消除 Blocks

## 题意翻译

有n个带有颜色的方块，没消除一段长度为x的连续的相同颜色的方块可以得到x^2的分数，让你用一种最优的顺序消除所有方块使得得分最多。

输入格式

第一行包含测试的次数t（1≤t≤15）

每个案例包含两行。第一行包含整数n（1≤n≤200），即框数。第二行包含n个数，代表每个盒子的颜色。数字的大小1~n内。
