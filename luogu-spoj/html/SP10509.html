题目描述

Maricruz have a lot of cards, she always uses her cards to build pyramids as shown in the following image: ![Cards](../../content/francky:cards "Cards") A pyramid card of 3 levels. She always wonder how many cards does she need to make a pyramid card of **N** levels. Your task is to answer that question.

输入输出格式

输入格式


The first line of the input contains an integer 1 <= **T** <= 1,000. Each of the following **T** lines will have an integer 1 <= **N** <= 1,000,000.

输出格式


For each case, output a single line consisting of the number of cards needed to build a pyramid card of level **N** modulo 1,000,007.

Example
2
3
7

输出样例 #1

Example
