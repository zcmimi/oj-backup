NUMTRY - Number Theory

题目描述

f(n) and g(n) are two functions defined as following : f(n) = ∏( p $ _{i} $ $ ^{2ei+1} $ +1 ), where p $ _{i} $ is prime factor of n and e $ _{i} $ is highest power of p $ _{i} $ in n. g(n) = Σ( n/gcd(n,i) ); 1 <= i <= n For a given value of n, you have to compute \[f(n)/g(n)\] % 1000000007.

输入输出格式

输入格式


First line has T ( <= 10000 ), next T lines has 2 <= n <= 10^12.

输出格式


\[f(n)/g(n)\] % 1000000007 for each test case.

输入输出样例

输入样例 #1

2\n2\n4

输出样例 #1

\n3\n3 \n\n
