WORMS - Play with Digit Seven

题目描述

输入输出格式

输入格式


The input begins with an integer _t_(_t_<= 210), the number of test cases._t_ test cases follow. For each test case, the first and only line contains an integer _n_ (1<=_n_<=500).

输出格式


For each test case the output consists of one line that contains the answer.
