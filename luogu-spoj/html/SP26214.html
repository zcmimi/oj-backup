PFACTORS - Pisano Factors

题目描述

Given an integer n. Find how many integers c are there such that their pisano period is a factor of n. 1 <= c <= 10^5 There are mutiple test cases.

输入输出格式

输入格式


The first line contains number of test cases, 1 <= t <= 100 Next t lines contain an integer n each. 1 <= n <= 10^9

输出格式


Output the answer to each test case on a separate line.

输入输出样例

输入样例 #1

3\n6\n9\n10\n\n

输出样例 #1

3\n2\n2\n&nbsp;\n
