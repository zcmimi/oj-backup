BFTAB - BF Vector

题意翻译

## 题意 在一个长小于100的字符串中寻找第（N+1）个字符 ## 输入 第一行一个字符串 第二行一个二位数，会补0 ## 输出 一个字符，如题意 ## 提示 必须用Brainf**k语言（虽然LG并不行）

题目描述

Your task is to put in the output n char of the read input. You have to use BF

输入输出格式

输入格式


\- first line have x chars (1<=x<=99) and ends with a line feed (10) \- second line is one number n (00<=n<=99)

输出格式


Its only one char

输入输出样例

输入样例 #1

01234567
02

输出样例 #1

2
