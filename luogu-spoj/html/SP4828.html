ZSEQ - ZSequence

题目描述

You will be given a sequence A containing **N** positive integers, **a $ _{1} $** , **a $ _{2} $** , ..., **a $ _{N} $** . Let **S(i, j)** = **a $ _{i} $** + **a $ _{i + 1} $** + ... + **a $ _{j} $** , if i <= j. You should find **K** - 1 indexes, **m $ _{1} $** < **m $ _{2} $** < ... < **m $ _{K - 1} $** such that **lb $ _{1} $** <= **S(1, m $ _{1} $ )** <= **ub $ _{1} $** , ..., **lb $ _{i} $** <= **S(m $ _{i - 1} $ + 1, m $ _{i} $ )** <= **ub $ _{i} $** and **lb $ _{K} $** <= **S(m $ _{K - 1} $ + 1, N)** <= **ub $ _{K} $** . If the case of multiple solution, print the first lexicographically.

输入输出格式

输入格式


The first line of the standard input contains two space-separated integers **N** (2 <= **N** <= 5 000) and **K** (1 <= **K** - 1 <= **N** - 1). Next **N** lines contain integers **a $ _{1} $** , **a $ _{2} $** , ..., **a $ _{N} $** , respectively, 1 <= **a $ _{i} $** <= 10 $ ^{5} $ . **i**-th of the next **K** lines contain integers **lb $ _{i} $** and **ub $ _{i} $** , 1 <= **lb $ _{i} $** <= **ub $ _{i} $** <= 10 $ ^{9} $ .

输出格式


On the first line of the standard ouput you should print space-separated **K** - 1 indices of the solution as already explained. If such solution does not exist, you should print only one integer -1. **Note:** Memory limit is 16MBs.

输入输出样例

输入样例 #1

\n4 3\n1\n2\n3\n4\n1 3\n2 4\n3 10\n\n

输出样例 #1

1 2\n\n\nInput:\n4 3\n1\n2\n3\n4\n1 3\n2 4\n3 4\n\nOutput:\n2 3\n\n\nInput:\n4 3\n1\n2\n3\n4\n1 3\n2 4\n3 3\n\nOutput:\n-1
