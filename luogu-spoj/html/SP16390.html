题目描述

A television center has received N requests from their customers, which are labeled from 1 to N. The "i" customer request contains the beginning and ending of time of the requested program which are labeled as s(i) and t(i) respectively. The time for switching between two programs is not consistently. Finding the minimum channels need to be used so that all of the customers are satisfied?
