题目描述

Given the number, you are to answer the question: "Is it prime?" **Solutions to this problem can be submitted in C, C++, Pascal, Perl, Python, Ruby, Lisp, Hask, Ocaml, Prolog, Whitespace, Brainf\*\*k and Intercal only.**

输入输出格式

输入格式


_t_ – the number of test cases, then _t_ test cases follows. \[t <= 500\] Each line contains one integer: _N_ \[2 <= _N_ <= 2^63-1\]

输出格式


For each test case output string "YES" if given number is prime and "NO" otherwise.

输入输出样例

输入样例 #1

5
2
3
4
5
6

输出样例 #1

YES
YES
NO
YES
NO
