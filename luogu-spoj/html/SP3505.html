CPRIME - Prime Number Theorem

题目描述

[English](/problems/CPRIME/en/) [Vietnamese](/problems/CPRIME/vn/)In number theory, the Prime Number Theorem describes the asymptotic distribution of prime numbers. Let π(x) be the number of prime numbers not greater than x. The Prime Number Theorem states that: ![](https://cdn.luogu.com.cn/upload/vjudge_pic/SP3505/5df2faa0375cd2849eed63bd256e44fd30b84b7c.png)Your task is to write a program to verify how well the Prime Number Theorem can estimate π(x). To be more precise, for a given x, you have to calculate the percent error |π(x) - x/lnx| / π(x) %.

输入输出格式

输入格式


The input contains several test cases (no more than 1000). Each test case contains a value of x (2 ≤ x ≤ 10 $ ^{8} $ ) given in one line. A number 0 terminates the input.

输出格式


For each value of x, output the percent error of the estimation of π(x), rounded to 1 decimal digit.

10000000

5

0

输出样例 #1
