BOKAM143SOU - Checking cubes.

题意翻译

给定整数**N**。找到用不超过五个立方数表示**N**的方法的数量

题目描述

Given a integer \*\*N\*\*. Find number of posiible ways to represent \*\*N\*\* as a sum of at most five cubes.

输入输出格式

输入格式


First line contains N. 1<=N<=125000.

输出格式


Output the result

输入输出样例

输入样例 #1

64

输出样例 #1

2
