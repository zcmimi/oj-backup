LPRIME - Primes of Lambda

题目描述

[Lambda](../../../users/tonyxty/) checks primality in a weird way. He checks the following two conditions. - All the digits of the number in the decimal system are primes or one, namely 1, 2, 3, 5 or 7. - It isn't a multiple of 2, 3, 5, 7, 11 or 47 (Why 47? I don't know). In order to estimate the accuracy of his approach, he asks you to calculate the number of decimal integers of a specific length that satisfy the conditions.

输入输出格式

输入格式


The first and only line contains an integer _n_, denoting the length of integers.

输出格式


The only line contains the answer **modulo 9973**.

输入输出样例

输入样例 #1

\n1\n\n

输出样例 #1

\n1\n\nInput:\n2\n\nOutput:\n8\n\nInput:\n4\n\nOutput:\n182\n\nInput:\n1000000000\n\nOutput:\n4589\n\n
