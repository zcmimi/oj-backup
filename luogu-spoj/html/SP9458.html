GHOSTS - Ghosts having fun

题目描述

Ghost are living in big castle with K rooms. As they have around few hundred years and very tired, they decided to buy teleports. Every teleport can work only in one way ( to prevent collision ). Ghosts have decided which teleports they want to build and in which order they should be built. King of ghosts, Bob, asked you to check list of teleports and decide which of them do not build. He don't want ghosts having fun in infinite ride with teleports.

输入输出格式

输入格式


In first line - number K<=1000, amount of rooms in castle. In next line - number T<=300000 of teleports. In next T lines a,b<=K Rooms are numbered 1..K

输出格式


Print teleports which should not be built. End test case with 0 0

输入输出样例

输入样例 #1

4\n5\n2 4\n4 3\n3 2\n1 2\n3 1

输出样例 #1

3 2\n3 1\n0 0
