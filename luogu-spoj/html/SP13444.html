CNT_LUCK - Counting Lucky Numbers

题目描述

Find out how many numbers between a and b (inclusive) when represented as binary numbers have sum of digits lucky. A number is lucky if it's decimal representation contains digits 4 and 7 only. eg. 4, 7, 47 , 77 etc. where as 14 , 41 etc. are not. Note that 0 <= a <= b <= 10^19.

输入输出格式

输入格式


T: no of test cases T<=10^5 Next T lines have a and b in every line. a <= b

输出格式


for every test case output as described in problem statement

输入输出样例

输入样例 #1

2
15 15
63 63

输出样例 #1

1
