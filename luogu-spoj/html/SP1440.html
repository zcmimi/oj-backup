ARCTAN - Use of Function Arctan

题目描述

It's easy to know that arctan(1/2)+arctan(1/3)=arctan(1). Given number A, write a program to calculate the minimum sum B+C. A,B and C are all positive integers and satisfy the equation below: arctan(1/A)=arctan(1/B)+arctan(1/C)

输入输出格式

输入格式


The first line contains a integer number T(about 1000). T lines follow,each contains a single integer A, 1<=A<=60000.

输出格式


T lines,each contains a single integer which denotes to the minimum sum B+C.

输入输出样例

输入样例 #1

1
1

输出样例 #1

5
