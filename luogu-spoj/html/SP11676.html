CPAIR2 - Counting diff-pairs

题意翻译

## 题意简述： 给你一个长度为n的序列a,满足a[i]<=100000. 共m次询问， 每次询问给你一个l和r， 问有多少对数(i,j)在区间[l,r]内 满足l<=i=k. n<=50000,m<=50000

题目描述

You are given sequence A of N integers. You are also given integer K and M queries. Each query consists of two integers l, r. For each query output number of pairs i, j such that l <= i < j <= r and abs(A\[i\] - A\[j\]) >= K. Indexing starts with 1. N <= 50000 M <= 50000 1 <= A\[i\] <= 100000 NOTE: All tests are randomly generated.

输入输出格式

输入格式


First line of input contains integers N, M, K in this order. Second line contains N integers representing array A. Next M lines describe queries.

输出格式


Output answer for each query.

输入输出样例

输入样例 #1

3 1 2\n1 2 3\n1 3\n

输出样例 #1

1
