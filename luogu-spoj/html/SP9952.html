GUANGGUN - 111&#8230;1 Squared

题目描述

We call an integer _Sticks_ if its decimal representation contains only digit 1. Let S(**n**) be the sticks with **n** digits, find the sum of digits of S(n) $ ^{2} $ For example, S(9) $ ^{2} $ =12345678987654321, its sum of digits is 81.

输入输出格式

输入格式


Each line contains an integer represents **n**(1 ≤ **n** ≤ 10 $ ^{18} $ ). Input is terminated by EOF, and contains at most 100 lines.

输出格式


For each **n**, print an integer represents the answer.

输入输出样例

输入样例 #1

9
10

输出样例 #1

81
82
