NOVICE62 - Match the words

题意翻译

题目描述 您可能会在第二列中遇到类似“匹配后续”单词及其同义词等问题。特德偶然发现了一个这样的问题。在英语方面表现不佳，他试着猜测答案。虽然他猜到了，但他想到了他在猜测时有多么不走运。 在数学方面也很弱，他需要你的帮助来完成一项稍微不同的任务。找到完成这个问题的解决方案的总方式，在这个问题中他没有得到正确的匹配。 注意：在匹配问题中，有两列单词。每列包含N个单词。第一列中的单词必须与第二列中的单词匹配。该问题的有效解决方案要求第一列中的每个单词仅与第二列中的一个单词匹配，反之亦然。 输入格式 第一行包含单个整数T（1 <= T <= 1000），表示测试用例的总数。每个测试用例包含一个整数N（1 <= N <= 1000000）。

题目描述

You may have come across questions like, Match-The-Following words with their synonyms in the second column and alike. Ted also stumbled upon one such question. Being weak at english, he tried guessing the answers. While he guessed, he thought about how unlucky he was at guessing. Being also weak at math, he needs help from you for a slightly different task. Find the total number of ways of completing the solution to this question in which he gets none of the matches correct. **Note:** In the match-the-following problem, there are two columns of words. Each column contains N words. Words in the first column have to be matched with words in the second column. A valid solution to this question requires every word in the first column to be matched with only one word in the second column and vice-versa.

输入输出格式

输入格式


First line contains a single integer T(1 <= T <= 1000) denoting the total number of test cases. Each test case contains a single integer N (1 <= N <= 1000000).

输出格式


For each test case, print a single integer the required number of ways modulo 1000000007

输入输出样例

输入样例 #1

3
1
2
7

输出样例 #1

0
1
1854
