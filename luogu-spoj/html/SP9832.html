MIFF - Matrix inverse

题目描述

Let p a prime number. A set F $ _{p} $ ={0,1,...,p-1} equipped with the mod p addition and multiplication is a finite field. In this problem you have to compute the multplicative inverse of some F $ _{p} $ valued (quadratic) matrices. The input consists of blocks. The structure of a block is: n p A $ _{11} $ ...A $ _{1n} $ ... A $ _{n1} $ ...A $ _{nn} $ where p is a prime number, 1
