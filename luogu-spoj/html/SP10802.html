TETRAHRD - Sum of Tetranacci numbers

题目描述

The sequence of Tetranacci numbers is defined as follows: a $ _{n} $ = a $ _{n-1} $ + a $ _{n-2} $ + a $ _{n-3} $ + a $ _{n-4} $ with a $ _{0} $ = a $ _{1} $ = a $ _{2} $ = 0 and a $ _{3} $ = 1.

输入输出格式

输入格式


Input starts with a positive integer t

输出格式


Calculate a $ _{m} $ + a $ _{m+1} $ + ... + a $ _{n} $ and print the result modulo 1000000007.

输入输出样例

输入样例 #1

2
1 2
1919 5393

输出样例 #1

0
66616
