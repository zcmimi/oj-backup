DIVREL - Divisibility Relation

题目描述

[English](/problems/DIVREL/en/) [Vietnamese](/problems/DIVREL/vn/)Given n positive integers. Your task is to select a maximum number of integers so that there are no two numbers a, b in which a is divisible by b.

输入输出格式

输入格式


- Line 1: n (1 ≤ n ≤ 200). - Line 2: n positive integers a $ _{1} $ , a $ _{2} $ , ..., a $ _{n} $ (1 ≤ a $ _{i} $ ≤ 10 $ ^{9} $ ).

输出格式


- Line 1: k, the maximum number of integers that can be selected. - Line 2: k selected integers.

输入输出样例

输入样例 #1

8
1 2 3 5 6 8 7 9

输出样例 #1

5
5 6 8 7 9

Input
4
2 3 2 3

Output
2
2 3
