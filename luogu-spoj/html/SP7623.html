DIVISER9 - Divisors VI

题目描述

Find the the smallest integer with exactly **N** odd divisors, where each divisor is greater or equal to 1 !!! For example for 3 odd divisors, 9 (factors 1, 3, 9) is minimum.

输入输出格式

输入格式


Each line contain **N (0 < N < 10 $ ^{14} $ )**. There are 100 inputs total.

输出格式


One answer on each line for each **N**, **Modulo 1000000007.**

输入输出样例

输入样例 #1

\n3\n1024\n\n

输出样例 #1

\n9\n97947700\n\n
