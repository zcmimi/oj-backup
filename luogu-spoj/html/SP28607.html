Given an array **A** of **k** integers (not necessarily distinct), we define the _descending alternating sum_ of this array, denoted **F(A)** the following way. First, we sort the array in descending order. Suppose the elements, after sorting, are **A $ _{1} $** >= **A $ _{2} $** >= ... >= **A $ _{k} $** respectively. Then the descending alternating sum of array **A** is **F(A) =** **A $ _{1} $ - A $ _{2} $ + A $ _{3} $ - ... + (-1) $ ^{k+1} $ A $ _{k} $** . For example, if **A = \[5, -3, 8, 2, 0, -5\]** then after sorting it in descending order, we find **A = \[8, 5, 2, 0, -3, -5\]**. So the descending alternating sum of this array is **8 - 5 + 2 - 0 + (-3) - (-5) = 7**. In particular, the descending alternating sum of an empty array is 0.
