WILLITST - Will it ever stop

题意翻译

题目：输入n，然后判断该代码是否会停止运行。 ```cpp while n > 1 if n mod 2 = 0 then n:=n/2 else n:=3*n+3 ``` 输入格式：一个整数n。 输出格式：若该代码会停止运行输出“TAK”。否则输出“NIE”。 感谢@qq1010903229 提供翻译

题目描述

When Bob was in library in University of Warsaw he saw on one of facades caption :"Will it ever stop?" and below some mysterious code: ``` while n > 1 if n mod 2 = 0 then n:=n/2 else n:=3*n+3 Help him finding it out ! ```

输入输出格式

输入格式


In first line one number n<=10^14.

输出格式


Print "TAK" if program will stop, otherwise print "NIE"

输入输出样例

输入样例 #1

4

输出样例 #1

TAK
