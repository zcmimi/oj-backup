MDST - Minimum Diameter Spanning Tree

题目描述

Solve the minimum diameter spanning tree problem for the simple graphs. For a given list of adjacent vertices of a graph G find the minimum diameter spanning tree T and write down the diameter of this tree diam(T). Each graph has only one connected component, so there is at least one spanning tree, which connects all the vertices.

输入输出格式

输入格式


t \[the number of test graphs\] Graph: _n_ \[1 <= _n_ <= 1000 the number of graph vertices\] _i m v $ _{1} $ v $ _{2} $ ... v $ _{m} $_ \[the list of _m_ adjacent vertices to vertex _i_\]

输出格式


For each test case output: _d_ \[diameter of the minimum diameter spanning tree\]
