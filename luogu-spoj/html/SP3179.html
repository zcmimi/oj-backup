DPEQN - Congruence Equation

题目描述

[English](/problems/DPEQN/en/) [Vietnamese](/problems/DPEQN/vn/) Given a congruence equation: ``` a1x1 + ... a2x2 + ... + anxn = b (mod m) ``` In which, a $ _{1} $ , a $ _{2} $ , ..., a $ _{n} $ , b and m are positive integer constants; x $ _{1} $ , x $ _{2} $ , ..., x $ _{n} $ are unknowns. Find a solution for this equation, or show that the equation has no solution.

输入输出格式

输入格式


First line: number of test cases. Each test case has the following form: - Line 1: n (1 ≤ n ≤ 100) - Line 2: n integers a $ _{1} $ , a $ _{2} $ , ..., a $ _{n} $ (1 ≤ a $ _{i} $ ≤ 10 $ ^{8} $ ) - Line 3: b, m (1 ≤ b, m ≤ 10 $ ^{8} $ ) Each test case is separated by a blank line.

输出格式


For each test case, if the equation has no solution, print "NO". Otherwise, print n integers x $ _{1} $ , x $ _{2} $ , ..., x $ _{n} $ (0 ≤ x $ _{i} $ < m) that is one solution to the equation.

输入输出样例

输入样例 #1

2

2
4 6
6 10

2
4 6
3 8

输出样例 #1

1 2
NO
