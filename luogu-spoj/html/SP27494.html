HAPPINESS - Happiness

题意翻译

有一个$n$个元素的数组$a[1]$、$a[2]$、$a[3]$……$a[n]$。 现在你需要回答一些询问。 在每个询问中，都会给你一个区间$[l,r]$。对于这个区间，你需要输出区间$[l,r]$中所有元素的"幸福度"之和。 元素$a[i]$在区间$[l,r]$之间的幸福度定义为：子区间$[l_j,r_j]$之间的最小值等于$a[i]$的数目，其中,$l<=l_j<=i$且$i<=r_j<=r$。 要求输出所有$[l,r]$之间元素幸福度的总和。 本题多组数据。 输入的第一行包含数据组数$t$。每组数据的第一行包括两个数字，$n$和$m$。$n$是数组$a$中的元素数，$m$是询问数。 下一行包含$n$个整数，为数组$a$。 接下来的$m$行包含两个整数，$l$和$r$。 对于每组数据，在第一行以"case x:"的格式输出数据组号，其中$x$是组号。 在接下来的m行中，输出该数组每个对应的$[l,r]$区间中所有元素的幸福度之和。

题目描述

You are given an array on N elements a\[1\], a\[2\], a\[3\], …. , a\[N\]. Now you will have to answer some queries. In every query, you will be given an interval, \[l, r\]. For this interval you have to print the total summation of happiness of all the elements of the given array between the interval \[l, r\]. The happiness of an element a\[i\] between interval \[l, r\] is: the number of sub-array \[lj , rj\] where the minimum value between \[lj, rj\] is equal to a\[i\]. Here, _l <= lj <= i & i <= rj <= r._ Now, you have to print the total summation of happiness of all the elements between \[l,r\].

输入输出格式

输入格式


The first line of the input contains the number of test cases T. The first line of each test case contains two numbers, N & M. N is the number of elements in array a and M is the number of queries you need to perform. The next line contains N integers, the array a : a\[1\], a\[2\], a\[3\], … a\[N\]. Next M lines contains two integers, l & r.

输出格式


For each test case, you need to print the case number on the first line in this format: Case X: where X is the case number. In the next M lines, you need to print the total summation of happiness of all the elements between \[l, r\] of the given array.

输入输出样例

输入样例 #1

2\n5 2\n1 3 2 4 3\n1 3\n2 4\n5 2\n5 3 7 6 8\n1 4\n2 5&nbsp;

输出样例 #1

Case 1:\n6\n6\nCase 2:\n10\n10
