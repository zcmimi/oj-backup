DIVFACT4 - Divisors of factorial (extreme)

题目描述

Find the number of divisors of **_N_**! (factorial) very fast !

输入输出格式

输入格式


The first line contains an integer _**T**_, the number of test cases. Each line of the next **_T_** lines contains two integers _**N**_ and **_M_**.

输出格式


For each line, output a single line containing the number of divisors of **_N_**! (modulo _**M**_).

输入输出样例

输入样例 #1

3
10 989
10000 999989
10000000000 999999999989

输出样例 #1

270
616797
40946947081
