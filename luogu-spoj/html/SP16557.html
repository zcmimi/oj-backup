FORMAT - HTML Formatting

题目描述

.example-table { text-align: left; margin-top: 10px; width: 100%; } .example-table td { vertical-align: top; } .section { margin-top: 19px; margin-bottom:19px; } .paragraph { text-align: left; margin-top: 10px; margin-bottom: 10px; } .paragraph ul, .paragraph ol { margin-top: 3px; margin-bottom: 3px; }

输入输出格式

输入格式


There will be a sequence of 1 to 20,000 numerals. Each numeral denotes the formatting at that position. - 0 - no formatting - 1 - bold - 2 - italic - 3 - bold and italic - 4 - underlined - 5 - bold and underlined - 6 - italic and underlined - 7 - bold, italic, and underlined (There is no need to actually specify the content, since that will not affect the answer.) The example earlier would be given as 333333220.

输出格式


The minimum number of **< b >**, **< / b >**, **< i >**, **< / i >**, **< u >**, and **< / u >** tags required.
