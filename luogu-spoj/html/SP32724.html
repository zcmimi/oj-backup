THREENUMBERS - nth number

题目描述

Given two numbers a and b, you have to find n-th number which is divisible by a or b.

输入输出格式

输入格式


First line contains an integer T (<= 100000), denoting number of test cases. Second line contains 3 positive integers a, b and n. (a, b <= 10000, n <= 1000000000)

输出格式


Print n-th number in a new line.

输入输出样例

输入样例 #1

1
2 3 10

输出样例 #1

15
