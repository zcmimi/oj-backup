题意翻译

## 题目描述 ##### 我很忙 你确实是个大忙人，你有很多活动安排，而你的目标是尽可能多的进行活动。 如果你与人约会，你就不能参加编程竞赛(NOIP?)也不能看电影；如果你玩[DotA](https://baike.baidu.com/item/dota/32506)，你就不能复习明天要考试的内容；如果你复习明天要考试的内容，你就不能安然入睡。 你可以看电影，玩DotA，安静地睡觉 or 与人约会，玩DotA并安静地睡觉 ## 输入格式 第一行包含整数T，即测试样例的数量。对于每个测试用例，第一行由整数N（活动数）组成。然后接下来的N行包含两个整数m和n，即每个活动的开始和结束时间。 1 <= T <= 10 1 <= N <= 100000 0 <= start
