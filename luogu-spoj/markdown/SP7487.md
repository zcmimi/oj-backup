# FLIB - Flibonakki

## 题目描述

G(n) is defined as

G(n) = G(n-1) + f(4n-1) , for n > 0

and G(0) = 0

f(i) is ith Fibonacci number. Given n you need to evaluate G(n) modulo 1000000007.

## 输入输出格式

### 输入格式

First line contains number of test cases t (t

### 输出格式

For each test case print G(n) modulo 1000000007.

## 输入输出样例

### 输入样例 #1

```cpp
2
2
4
```


### 输出样例 #1

```cpp
15
714
```
