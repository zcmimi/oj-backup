# MAXXOR - Find the max XOR value

## 题目描述

You have two integers **L and R**, and you are required to find the max xor value of a and b where L <= a <= R and L <= b <= R

## 输入输出格式

### 输入格式

Two integers in a line. L, R <= 1e9

### 输出格式

One integer, the answer

## 输入输出样例

### 输入样例 #1

```cpp
1 10
```


### 输出样例 #1

```cpp
15
```
