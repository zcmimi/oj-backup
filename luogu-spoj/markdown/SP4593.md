# ARITHSEQ - Arithmetic sequence

## 题目描述

[English](/problems/ARITHSEQ/en/) [Tiếng Việt](/problems/ARITHSEQ/vn/) Given an arithmetic sequence a, a + d, a + 2\*d, ..., a + n\*d.

## 输入输出格式

### 输入格式

\- A line containing a, d, n, x.

### 输出格式

\- The result found.

## 输入输出样例

### 输入样例 #1

```cpp
10 5 2 0
```


### 输出样例 #1

```cpp
2
```
